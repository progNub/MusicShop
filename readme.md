Для запуска проекта в докер контейнере необходимо иметь установленную программу докер.


Для запуска проекта нужно: 
1. Переименовать файл .env_example в файл .env.
2. Установить все необходимые переменные окружения, что находятся в файле .env.
3. Находясь в корневой директории прописать 'docker-compose up --build -d'
